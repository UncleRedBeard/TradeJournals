# The Forge And Shop

Documents the workshop buildout and restoration work, with emphasis on reclaimed timber, durable work surfaces, tool storage, material reuse, and the workspace practices that support the broader preservation portfolio.

## Suggested Journal Topics

- reclaimed timber sourcing, milling, and preparation
- workbench construction and shop fixtures
- layout decisions for woodworking, metalwork, and mechanical restoration
- lighting, power, dust control, and utility integration
- tool restoration, storage, and maintenance
