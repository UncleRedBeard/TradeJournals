# Machine Trade Journals

Use this folder for scooter, motorcycle, and mechanical restoration journals.
Organize entries by machine and system so the work can be explained as careful
diagnosis, preservation, repair, and reassembly rather than a simple
before-and-after gallery.

## Journals

- [1964 Vespa Restoration](1964_vespa_restoration.md) - craftsman-first
  restoration archive for bodywork, rust, wiring, steering, paint prep, and
  two-stroke scooter preservation evidence.
- [Bikes - Honda Chopper](bikes_honda_chopper.md) - custom motorcycle archive
  for the 1974 Honda CB750 SOHC engine and 1977 Santee chopper frame.
- [Bikes - 1982 Honda CB750 DOHC](bikes_1982_honda_cb750_dohc.md) - factory-era
  Honda service record focused on intake, carburetion, and right-side engine
  evidence.
- [Bikes - 2004 KLR 650](bikes_2004_klr_650.md) - dual-sport utility journal
  for rack, luggage, exhaust, tire, and road/trail readiness evidence.
- [Iron Butterfly Cycle Works](iron_butterfly_cycle_works.md) - machine/shop
  archive journal for motorcycle, fabrication, parts-cleaning, and cycle-work
  evidence.
