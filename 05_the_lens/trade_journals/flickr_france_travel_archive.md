# Flickr: France Travel Archive

## Purpose

Catalog the France Flickr travel albums as a grouped lens archive. This entry keeps the Flickr source material together as a coherent travel/photo sequence while preserving each album's public URL, album ID, and API-visible photo count for future image-level review.

## Archive Context

This journal belongs in `05_the_lens` because the albums document a travel-photo body of work rather than a single machine, room, or shop project. The group should be reviewed as a connected sequence across cameras, locations, and visual approaches.

Existing standalone Flickr sidecars for `Roqueburne-Cap-Martin`, `La Turbie`, and `Toulon` remain valid catalog entries. This grouped journal gathers the broader France Flickr set so the inventory has one coherent home for the remaining travel albums.

## Working Thesis

This archive is a France travel-photo body of work organized around place, movement, and camera character. The first review pass should identify which albums serve as location records, which images carry stronger standalone photographic value, and which photos best show the visual behavior of the camera/film combination.
